# 产品需求文档 — 食话实说

## 产品定位
面向普通消费者的移动端食品配料表与宣传比对工具。帮助消费者在超市/网购时快速识别虚假宣传，避免被营销话术误导。

## 目标用户
- 不懂配料表的普通消费者
- 关注食品安全但对成分不敏感的人群
- 移动端使用为主（手机浏览器）

## 核心功能

### F1：三种输入方式
| 方式 | 描述 | 优先级 |
|------|------|--------|
| 拍照识别 | 拍产品正面+配料表照片，OCR提取文字 | P0 |
| 手动输入 | 打字输入商品名和配料表 | P0 |
| 粘贴链接 | 粘贴电商商品页URL，自动抓取信息 | P1 |

### F2：AI智能比对
- 提取商品名中的暗示性关键词（如"100%"、"纯"、"无糖"、"天然"）
- 对照配料表逐项检查
- 识别规则：
  - 商品名暗示单一成分但配料含多成分 → 预警
  - 宣传"无糖"但配料含糖类 → 预警
  - 宣传"天然/无添加"但含香精/色素/防腐剂 → 预警
  - 果汁饮料第一配料是水而非果汁 → 预警

### F3：结果展示（简洁模式）
- 只展示有问题的项目
- 红色高亮预警
- 一句话总结建议
- 风险等级标识（高/中/低/无）

### F4：辅助功能
- OCR结果可人工校对编辑
- 分析结果缓存（localStorage）
- PWA支持（添加到手机主屏幕）

## 非功能需求
- 移动端优先，最大宽度480px
- 微信浏览器兼容
- 加载时间 < 3秒（含OCR+AI分析）
- 浅蓝色主色调
